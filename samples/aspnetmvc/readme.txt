 1. В MSSQL создать базу данных с названием MvcSample. Создать пользователя с именем MvcSample и паролем MvcSample, который будет иметь все права на эту базу данных.
 2. Открыть MvcSample.sln в VS2012
 3. Собрать проект, при этом должны скачаться все необходимые библиотеки через Nuget
 4. Запустить migrate.bat
 5. Запустить в классе LiveDbTests тест AddUser
 6. Запустить проект, в качестве логина ввести login и пароль 123
 7. Если вход прошел успешно, то должна появится надпись "ASP.NET MVC Sample + ByndyuSoft.Infrastructure"
