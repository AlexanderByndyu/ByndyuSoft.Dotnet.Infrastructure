 1. В MSSQL создать базу данных с названием Mvc4Sample. Создать пользователя с именем Mvc4Sample и паролем Mvc4Sample, который будет иметь все права на эту базу данных.
 2. Открыть Mvc4Sample.sln в VS2012
 3. Собрать проект, при этом должны скачаться все необходимые библиотеки через Nuget
 4. Запустить migrate.bat
 5. Запустить в классе LiveDbTests тест AddUser
 6. Запустить проект, в качестве логина ввести login и пароль 123
 7. Если вход прошел успешно, то должна появится надпись "ASP.NET MVC4 Sample + ByndyuSoft.Infrastructure"
